## 1. 环境准备

### 1.1 Node.js 安装

访问 [Node.js 官网](https://nodejs.org/) 下载并安装 LTS 版本。

安装完成后，打开终端验证：

![Node.js版本验证](/img/blog-tutorial/nodejs-version.svg)

```bash
node --version
npm --version
```

### 1.2 Git 安装

访问 [Git 官网](https://git-scm.com/) 下载并安装。

配置 Git 用户信息：
```bash
git config --global user.name "你的用户名"
git config --global user.email "你的邮箱"
```

## 2. Hexo 安装与初始化

### 2.1 安装 Hexo CLI

```bash
npm install -g hexo-cli
```

### 2.2 初始化博客

```bash
# 初始化 Hexo 博客
hexo init myblog
cd myblog
npm install

# 启动本地服务器
hexo server
```

![Hexo初始化过程](/img/blog-tutorial/hexo-init.svg)

访问 `http://localhost:4000` 查看默认博客。

### 2.3 博客文件夹结构

![博客文件夹结构](/img/blog-tutorial/folder-structure.svg)

```
myblog/
├── _config.yml          # 站点配置文件
├── package.json         # 依赖管理
├── scaffolds/           # 文章模板
├── source/              # 源文件目录
│   ├── _drafts/         # 草稿
│   └── _posts/          # 文章
├── themes/              # 主题目录
└── public/              # 生成的静态文件
```

## 3. Butterfly 主题安装

### 3.1 下载 Butterfly 主题

```bash
# 克隆主题到 themes 目录
git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

# 安装必要的渲染器
npm install hexo-renderer-pug hexo-renderer-stylus --save
```

![Butterfly主题安装](/img/blog-tutorial/butterfly-install.svg)

### 3.2 启用主题

修改根目录下的 `_config.yml`：
```yaml
theme: butterfly
```

### 3.3 复制主题配置文件

```bash
cp themes/butterfly/_config.yml _config.butterfly.yml
```

### 3.4 基础配置

编辑 `_config.butterfly.yml`：

```yaml
# 导航菜单
menu:
  首页: / || fas fa-home
  归档: /archives/ || fas fa-archive
  标签: /tags/ || fas fa-tags
  分类: /categories/ || fas fa-folder-open
  关于: /about/ || fas fa-heart

# 网站图标
favicon: /img/favicon.ico

# 头像
avatar:
  img: /img/avatar.jpg
  effect: false

# 网站副标题
subtitle:
  enable: true
  sub1: 欢迎来到我的博客
  sub2: 分享技术与生活
```

## 4. Live2D 看板娘配置

![Live2D看板娘配置](/img/blog-tutorial/live2d-config.svg)

### 4.1 安装 Live2D 插件

```bash
npm install hexo-helper-live2d --save
npm install live2d-widget-model-haru --save
```

### 4.2 配置 Live2D

在根目录 `_config.yml` 中添加：

```yaml
live2d:
  enable: true
  scriptFrom: local
  pluginRootPath: live2dw/
  pluginJsPath: lib/
  pluginModelPath: assets/
  tagMode: false
  debug: false
  model:
    use: live2d-widget-model-haru
  display:
    position: right
    width: 150
    height: 300
  mobile:
    show: false
  react:
    opacity: 0.7
```

### 4.3 可选模型

```bash
# 其他可选模型
npm install live2d-widget-model-shizuku --save
npm install live2d-widget-model-miku --save
npm install live2d-widget-model-wanko --save
```

## 5. Giscus 评论系统

### 5.1 准备工作

1. 创建 GitHub 公开仓库用于存储评论
2. 安装 [Giscus App](https://github.com/apps/giscus)
3. 在仓库设置中启用 Discussions 功能

### 5.2 获取配置参数

访问 [Giscus 官网](https://giscus.app/zh-CN) 获取配置参数。

### 5.3 配置 Giscus

在 `_config.butterfly.yml` 中配置：

```yaml
comments:
  use: Giscus
  text: true
  lazyload: true
  count: true
  card_post_count: true

giscus:
  repo: 你的用户名/评论仓库名
  repo_id: 你的仓库ID
  category_id: 你的分类ID
  theme:
    light: light
    dark: dark
  option:
    data-lang: zh-CN
    data-loading: lazy
    crossorigin: anonymous
```

## 6. 搜索功能配置

### 6.1 安装搜索插件

```bash
npm install hexo-generator-search --save
```

### 6.2 配置搜索

在根目录 `_config.yml` 中添加：

```yaml
search:
  path: search.xml
  field: post
  content: true
  format: html
```

在 `_config.butterfly.yml` 中启用：

```yaml
local_search:
  enable: true
  preload: false
  CDN:
```

创建 `source/css/custom.css`：

  height: 0;
  overflow: hidden;

  0% { background-position: 0% 50%; }

在 `_config.butterfly.yml` 中添加：

```yaml
inject:
  head:
    - <link rel="stylesheet" href="/css/custom.css">
```

## 9. 部署到 GitHub Pages

![GitHub Pages部署流程](/img/blog-tutorial/github-deploy.svg)

### 9.1 创建 GitHub 仓库

1. 在 GitHub 创建新仓库，命名为 `username.github.io`
2. 仓库设置为 Public

### 9.2 安装部署插件

```bash
npm install hexo-deployer-git --save
```

### 9.3 配置部署

在根目录 `_config.yml` 中配置：

```yaml
# URL
url: https://username.github.io
root: /

# Deployment
deploy:
  type: git
  repo: git@github.com:username/username.github.io.git
  branch: main
```

### 9.4 部署博客

```bash
# 清理并生成静态文件
hexo clean
hexo generate

# 部署到 GitHub
hexo deploy

# 或者一键部署
hexo clean && hexo g && hexo d
```

## 12. 常用脚本

创建便捷脚本 `scripts/blog.sh`：

```bash
#!/bin/bash

# 新建文章
if [ "$1" = "new" ]; then
    hexo new "$2"
    echo "文章 '$2' 创建成功！"
fi

# 本地预览
if [ "$1" = "server" ]; then
    hexo clean && hexo g && hexo s
fi

# 部署
if [ "$1" = "deploy" ]; then
    hexo clean && hexo g && hexo d
    echo "博客部署完成！"
fi
```

使用方法：
```bash
# 新建文章
./scripts/blog.sh new "文章标题"

# 本地预览
./scripts/blog.sh server

# 部署博客
./scripts/blog.sh deploy
```

## 13. 常见问题解决

### 13.1 部署失败

检查 SSH 密钥配置：
```bash
ssh-keygen -t rsa -C "your_email@example.com"
cat ~/.ssh/id_rsa.pub
```

### 13.2 主题更新

```bash
cd themes/butterfly
git pull
```

### 13.3 插件冲突

清理 node_modules：
```bash
rm -rf node_modules
npm install
```
