## 🔍 搜索与导航类

### 1. hexo-generator-searchdb
**功能**: 本地搜索数据生成
```bash
npm install hexo-generator-searchdb --save
```

**配置** (`_config.yml`):
```yaml
search:
  path: search.xml
  field: post
  content: true
  format: html
```

### 2. hexo-generator-sitemap
**功能**: 生成 sitemap.xml，提升 SEO
```bash
npm install hexo-generator-sitemap --save
```

**配置**:
```yaml
sitemap:
  path: sitemap.xml
  template: ./sitemap_template.xml
  rel: false
  tags: true
  categories: true
```

### 3. hexo-generator-robotstxt
**功能**: 生成 robots.txt
```bash
npm install hexo-generator-robotstxt --save
```

**配置**:
```yaml
robots:
  useragent: "*"
  disallow:
    - /js/
    - /css/
  allow:
    - /
  sitemap: /sitemap.xml
```

```yaml
  display_in_home: false
  params:
    maxCount: 5
    PPMixingRate: 0.0
    isDate: false
    isImage: false
    isExcerpt: false
```

**功能**: 代码压缩优化
```bash
npm install hexo-neat --save
```

**配置**:
```yaml
neat_enable: true
neat_html:
  enable: true
  exclude:

  enable: true
  exclude:
    - '*.min.css'
neat_js:
  enable: true
  mangle: true
  exclude:
    - '*.min.js'
```

## 📱 PWA 与性能类

### 1. hexo-offline
**功能**: PWA 离线支持
```bash
npm install hexo-offline --save
```

**配置**:
```yaml
# 在 _config.yml 中添加
offline_config:
  maximumFileSizeToCacheInBytes: 10485760
  staticFileGlobs:
    - public/**/*.{js,html,css,png,jpg,gif,svg,eot,ttf,woff,woff2}
  stripPrefix: public
  verbose: true
  runtimeCaching:
    - urlPattern: /*
      handler: cacheFirst
      options:
        origin: your-cdn-domain.com
```

### 2. hexo-generator-feed
**功能**: RSS 订阅支持
```bash
npm install hexo-generator-feed --save
```

**配置**:
```yaml
feed:
  enable: true
  type: atom
  path: atom.xml
  limit: 20
  hub:
  content:
  content_limit: 140
  content_limit_delim: ' '
  order_by: -date
  icon: icon.png
  autodiscovery: true
  template:
```

### 3. hexo-filter-optimize
**功能**: 自动优化图片和资源
```bash
npm install hexo-filter-optimize --save
```

**配置**:
```yaml
optimize:
  enable: true
  versioning: false
  css:
    enable: true
    bundle: true
    delivery: true
    inlines:
      enable: true
      threshold: 1024
  js:
    enable: true
    bundle: true
    delivery: true
  html:
    enable: true
  image:
    enable: true
    interlaced: false
    multipass: false
    optimizationLevel: 2
    pngquant: false
    progressive: false
```

## ⚠️ 注意事项

### 插件冲突
- `hexo-renderer-marked` 与 `hexo-renderer-kramed` 冲突
- 某些压缩插件可能与主题冲突
- Live2D 插件过多可能影响性能

### 性能考虑
- 不要同时安装过多相似功能插件
- 定期清理不使用的插件
- 注意插件的维护状态

### 兼容性
- 检查插件与 Hexo 版本兼容性
- 测试插件与主题的兼容性
- 备份配置文件再安装新插件
