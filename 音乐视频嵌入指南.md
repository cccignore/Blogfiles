# 音乐和视频嵌入指南

## 🎵 音乐平台嵌入方法

### 1. 网易云音乐

#### 获取嵌入代码：
1. 打开网易云音乐网页版
2. 找到想要分享的歌曲
3. 点击"生成外链播放器"
4. 复制iframe代码

#### 嵌入代码格式：
```html
<iframe frameborder="no" border="0" marginwidth="0" marginheight="0" width="330" height="86" src="//music.163.com/outchain/player?type=2&id=歌曲ID&auto=1&height=66"></iframe>
```

#### 参数说明：
- `id=歌曲ID`: 歌曲的唯一标识
- `auto=1`: 自动播放 (1=开启, 0=关闭)
- `type=2`: 播放器类型 (2=单曲, 0=歌单)

### 2. QQ音乐

#### 嵌入代码格式：
```html
<iframe frameborder="0" width="330" height="110" src="https://y.qq.com/n/ryqq/player?songid=歌曲ID&autoplay=0"></iframe>
```

### 3. Spotify

#### 嵌入代码格式：
```html
<iframe style="border-radius:12px" src="https://open.spotify.com/embed/track/歌曲ID?utm_source=generator" width="100%" height="152" frameBorder="0" allowfullscreen="" allow="autoplay; clipboard-write; encrypted-media; fullscreen; picture-in-picture" loading="lazy"></iframe>
```

## 📺 视频平台嵌入方法

### 1. 哔哩哔哩 (B站)

#### 获取嵌入代码：
1. 打开B站视频页面
2. 点击"分享" → "嵌入代码"
3. 复制iframe代码

#### 嵌入代码格式：
```html
<!-- 基础嵌入代码 -->
<iframe src="//player.bilibili.com/player.html?bvid=BV号&page=1" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="100%" height="400"></iframe>

<!-- 禁用自动播放（推荐） -->
<iframe src="//player.bilibili.com/player.html?bvid=BV号&page=1&autoplay=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="100%" height="400"></iframe>

<!-- 使用响应式容器（推荐） -->
<div class="responsive-video-container">
  <iframe src="//player.bilibili.com/player.html?bvid=BV号&page=1&autoplay=0" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true"></iframe>
</div>
```

#### 常用参数说明：
- `autoplay=0`: 禁用自动播放（推荐）
- `autoplay=1`: 启用自动播放
- `page=1`: 指定播放第几P
- `t=30`: 从第30秒开始播放

### 2. YouTube

#### 嵌入代码格式：
```html
<iframe width="100%" height="400" src="https://www.youtube.com/embed/视频ID" frameborder="0" allowfullscreen></iframe>
```

### 3. 腾讯视频

#### 嵌入代码格式：
```html
<iframe frameborder="0" width="100%" height="400" src="https://v.qq.com/txp/iframe/player.html?vid=视频ID" allowFullScreen="true"></iframe>
```

## 🎨 自定义样式

### 响应式设计
```html
<div style="position: relative; padding-bottom: 56.25%; height: 0; overflow: hidden;">
  <iframe src="视频链接" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;" frameborder="0" allowfullscreen></iframe>
</div>
```

### 音乐播放器居中
```html
<div style="text-align: center; margin: 20px 0;">
  <iframe frameborder="no" border="0" marginwidth="0" marginheight="0" width="330" height="86" src="音乐链接"></iframe>
</div>
```

## 📝 使用建议

### 1. 版权注意事项
- 仅用于个人学习交流
- 注明音乐/视频版权归原作者所有
- 避免商业用途

### 2. 用户体验优化
- 设置 `auto=0` 避免自动播放
- 合理控制嵌入内容数量
- 提供文字描述作为备选

### 3. 加载性能
- 使用 `loading="lazy"` 延迟加载
- 考虑使用封面图片+点击播放的方式

## 🔧 常见问题解决

### 1. 嵌入失败
- 检查链接是否正确
- 确认平台是否支持外链
- 尝试使用HTTPS链接

### 2. 移动端适配
- 使用响应式设计
- 测试不同屏幕尺寸
- 考虑移动端播放限制

### 3. 加载缓慢
- 优化iframe尺寸
- 使用CDN加速
- 考虑预加载策略
